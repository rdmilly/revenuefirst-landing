# Website Examples

A collection of website mockups and landing page examples by [Millyweb Development](https://millyweb.com).

## RevenueFirst.AI

Landing pages for [RevenueFirst.AI](https://revenuefirst.ai) - AI-powered sales automation for service businesses.

### Files

| File | Description |
|------|-------------|
| `revenuefirst-ai/coming-soon.html` | Original dark theme coming soon page |
| `revenuefirst-ai/v2-professional.html` | Professional blue/red business design (no images) |
| `revenuefirst-ai/v2-with-photos.html` | Full professional design with Unsplash stock photos |

### Design System

**V2 Professional Theme:**
- **Primary Color (Trust Blue):** `#1e3a5f` - Deep navy for headers, text, backgrounds
- **Accent Color (Power Red):** `#c41e3a` - CTAs, badges, highlights
- **Typography:** Playfair Display (headlines) + Inter (body)
- **Style:** Confident business professional, trust-inducing aesthetic

### Stock Photos

All images from [Unsplash](https://unsplash.com) - free for commercial use, no attribution required.

---

## Usage

These are static HTML files. Simply open in a browser or deploy to any web host.

```bash
# Local preview
open revenuefirst-ai/v2-with-photos.html

# Or serve with Python
python -m http.server 8000
```
